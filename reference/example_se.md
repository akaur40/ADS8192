# Example fission yeast SummarizedExperiment

RNA-seq count data from the fission yeast stress response experiment.
Includes sample metadata such as strain and minute.

## Usage

``` r
example_se
```

## Format

A SummarizedExperiment object with:

- assays:

  counts - raw count matrix

- colData:

  sample metadata including strain and minute

## Source

fission Bioconductor package
